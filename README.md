# react-native-knotapi

## Getting started

`$ npm install @millionscard/react-native-knotapi --save`

or 

`$ yarn add @millionscard/react-native-knotapi`

### Mostly automatic installation

`$ react-native link @millionscard/react-native-knotapi`
